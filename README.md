# Hotel Booking App

## 🧠 Project Overview
A web application for hotel booking.

## 🛠️ Technologies Used
- **Frontend:**
  - React.js
  - Bootstrap (CSS)
- **Backend:**
  - Node.js
  - Express.js
  - MongoDB
- **Others:**
  - Google Maps API
  - Git & GitHub

## 🚀 Installation & Running
git clone <repository-link>
cd HOTEL-BOOKING-APP
Install dependencies
npm install
# Start backend (in one terminal)
nodemon server.js
# Start frontend (in another terminal)
npm start
